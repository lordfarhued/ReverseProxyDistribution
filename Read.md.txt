# Система распределенного хранения и доставки конфигураций

## Описание решения

Система для управления конфигурациями reverse-proxy серверов в распределенной сети.

### Архитектура

1. **Master Server** (ASP.NET Core + SignalR)
   - Централизованное хранилище конфигураций (SQLite)
   - Real-time рассылка обновлений через WebSocket
   - REST API для управления

2. **Instance Agent** (Console App)
   - Подключается к Master Server
   - Получает обновления в реальном времени
   - Сохраняет конфигурации локально
   - Отправляет метрики

3. **Web UI** (HTML/JS)
   - Управление конфигурациями
   - Мониторинг инстансов
   - Блокировка IP

### Технологии

- **Backend**: C#, ASP.NET Core 8.0
- **Real-time**: SignalR (WebSocket)
- **Database**: SQLite + Entity Framework Core
- **Frontend**: HTML5, CSS3, Vanilla JavaScript

### Преимущества решения

✅ **Мгновенная доставка** - обновления через WebSocket за миллисекунды
✅ **Отказоустойчивость** - автоматическое переподключение агентов
✅ **Масштабируемость** - поддержка тысяч инстансов
✅ **Версионирование** - отслеживание версий конфигураций
✅ **Оперативное управление** - блокировка IP в 1 клик
✅ **Мониторинг** - метрики со всех инстансов

### Запуск

1. Запустить Master Server (F5 в Visual Studio)
2. Запустить агенты на каждом инстансе
3. Открыть Web UI в браузере
